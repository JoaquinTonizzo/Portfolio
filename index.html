Mis Proyectos

Lost Galaxian - Java

"Lost Galaxian" es un juego Java de aventuras espaciales donde los jugadores exploran galaxias perdidas enfrentándose a desafíos, enemigos y distintas amenazas con el objetivo de sobrevivir y eliminar a todos los adversarios en pantalla antes de que sea tarde.

Amazing Logistic - Java

Una aplicación que utiliza principios de programación orientada a objetos, como herencia y polimorfismo, abstracción e interfaces para optimizar el registro de pedidos, la gestión de paquetes y su entrega. Creando una solución eficiente, flexible y escalable.

Wordle Game - Python

"Wordle Game" es un proyecto en Python que ofrece una experiencia envolvente y divertida. Desafía tus habilidades lingüísticas al intentar adivinar palabras ocultas. Este proyecto refleja lacapacidad para crear aplicaciones interactivas y entretenidas en Python, un leguanje muy versártil.
